# string-mapping

This code solves genetic string mapping problem using best first search.
This implementation works for only two strings

Input format is followed as given in the assignment. But, k is limited to 2

Sample input: <br>

4            <br>
A, C, T, G  <br>
2          <br> 
ACTGTGA <br>
TACTGC <br>
3              <br>
0 2 2 2 1 <br>
2 0 2 2 1 <br>
2 2 0 2 1 <br>
2 2 2 0 1 <br>
1 1 1 1 0 <br>
#            <br>
